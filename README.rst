Dependencies?
~~~~~~~~~~~~~~~~

pyps is developed against `python`_ version 2.7.

Other dependencies are handled by `pip`_.

For building the docs from source with `sphinx`_, you will need the packages listed
under ``sphinx/requirements.pip``.
